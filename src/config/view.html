功能

界面调整

  • 自定义字体

    可通过候选框或手动输入字体家族名称启用此功能

  • 消息列表只显示头像

    消息列表将隐藏昵称/群名称/最新消息，同时会影响搜索界面，此功能会强行覆盖部分主题样式

  • 头像浮动

    模拟tg头像效果

  • 连续消息合并

    模拟tg消息效果 - 需要开启头像浮动

  • 保留时间

    启用 连续消息合并 后保留时间分隔 - 重载消息生效

  • 头像置底

    还原tg头像 - 需要开启头像浮动

  • 消息靠左

    自己发送的消息将靠左显示 - 重载消息生效

  • 禁用表情推荐

    修复官方设置有时无效的问题

  • 阻止拖拽多选消息

    避免误触发多选消息

  • 显示消息发送时间

    显示每一条消息的发送时间 - 重载消息生效

  • 显示完整日期

    会占用过多空间，建议配合 插入到用户名后方 使用

  • 插入到用户名后方

    在群组内时将时间插入到用户名后方

  • 快速关闭图片

    点击图片关闭查看窗口

  • 图片窗口全局拖拽

    旧版功能，针对过大图片无效，无法将窗口拖到屏幕外

  • 禁用表情中的GIF热图

    看个人喜好

  • 禁用小红点

    移除左侧按钮上的小红点

  • 未读气泡显示真实消息数量

    主窗口未读气泡将显示超过99条消息

  • 移除vip彩色昵称

    统一页面显示内容

  • 图片自适应窗口宽度

    解除图片大小限制 - 重载消息生效

  • 侧键返回

    按下鼠标返回侧键主窗口将回退到未选择聊天对象时的界面

  • 阻止 ESC 关闭窗口

    避免误操作

  • 窗口使用聊天头像作为图标

    不是任务栏图标，仅限 Windows 系统 - 重启生效

    右键图片搜索，%search%替换链接中搜索值

  • 保存到本地表情

    需要开启本地表情功能

  • 选项高亮

    人的本质是______ - 重载消息生效

  • 双击执行

    避免误触

  • 回复不带@

    移除回复时自带的@标记

  • 小程序分享转为url卡片

    部分元数据没有携带链接的小程序卡片将不会被转换

  • 记录离开时的位置

    进入时会自动回到上次离开的位置，但进入时可能会出现短暂闪烁。

  • 关键词提醒

    当消息中含有指定单词时会记录消息并提醒，当前只能在主窗口生效

    并生成一张类似tg中的预览卡片 - 重载消息生效

本地表情

  • 启用本地表情包

    可以通过文件夹分组表情包，按住Ctrl可多选表情，按住Alt可直接发送表情
    文件路径及名称不能存在特殊字符
    不推荐使用webp格式的动态表情，这会导致表情窗口异常卡顿

  • 移动功能位置

    将本地表情入口移动到功能列表左侧

  • 内存优化

    适用于加载表情过多的情况，推荐开启

  • 以图片形式发送

    表情会显示的更大

  • 历史表情

    记录最近使用过的指定数量表情在顶部显示

  • 历史表情保存数量

    自定义历史表情保存数量

  • 移入显示历史表情

    鼠标移入本地表情按钮后显示历史表情列表，需要开启 历史表情 功能

  • 快捷输入表情

    通过输入"/"唤起浮动表情列表，通过文件名进行筛选

  • 自动填充

    快捷输入表情 功能筛选只有一个表情时自动填充到输入框

  • 自定义快捷键

    自定义 快捷输入表情 功能的快捷键

    调整横排表情大小

消息后缀

  • 启用

    只有此选项开启的情况下，消息后缀才会生效

  • 后缀提示

    如果聊天对象匹配到了消息后缀，那么将会在发送按钮旁边显示后缀内容

  • 规则列表

    规则从上到下匹配，一次只会匹配一个规则

阻止撤回

  • 启用

    只有此选项开启的情况下，阻止撤回才会生效

  • 拦截自己的撤回操作

    本人发起的撤回也将被拦截

  • 拦截所有群组的撤回事件

    不开启时仅拦截最近打开过的聊天对象，启用后会占用更多资源，仅限高版本 - 重启生效

  • 持久化保存

    启用此选项后，所有撤回的消息将以base64格式保存在本地。如果担心消息泄露，请慎重开启。
    不启用此选项时撤回数据在重启后丢失，且不受 删除本地数据 功能影响。
    此选项仅保存启用后拦截到的撤回数据，开启前的内容将无法保留。
    如果关闭此选项，之前保存在本地的撤回消息也将不会加载。
    建议每隔一段时间清理一次本地数据

  • 自定义颜色

    调整撤回消息颜色 - 重载消息生效

  • 撤回提示颜色

    撤回提示标记显示颜色

  • 删除本地数据

    清除所有本地保存的撤回数据，当前保存有 0 条消息

  • 查看撤回消息

    只能查看持久化保存的撤回消息

  • 自定义背景

  • 覆盖侧边栏

    背景图片/视频内容覆盖窗口左侧功能栏

  • 窗口透明

    移除窗口背景层，建议搭配 More Materials 使用

  • 磨砂效果

    控制输入框等区域的磨砂效果

  • 增强可读性

    控制输入框等区域用于增强可见性的背景色

  • 重绘卡片

    重绘canvas卡片，统一消息样式 - 重载消息生效

  • 背景透明度

    背景图片/视频的透明度，范围为0-100
